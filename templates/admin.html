Solução de Problemas Comuns

Problema: A chave da API está incorreta, expirou ou não tem permissões.
Soluções:
  1. Verificar a chave da API:
    • Certifique-se de que copiou a chave completa do seu painel Evolution
    • Verifique se não há espaços extras no início ou fim
    • A chave não deve conter quebras de linha
  2. Regenerar chave:
    • Acesse o painel administrativo da sua Evolution API
    • Gere uma nova API Key
    • Copie e cole a nova chave aqui
  3. Testar manualmente: 
    curl --request GET \
  --url https://sua-api.com/instance/fetchInstances \
  --header 'apikey: sua-chave-aqui'
Dica: Se o erro persistir, entre em contato com quem administra seu servidor Evolution API.
